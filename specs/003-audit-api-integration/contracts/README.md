# Audit Log API Contracts

This directory contains the API contract specification for the audit log feature.

## Files

- `audit-api.yaml` - OpenAPI 3.0.3 specification for the audit log endpoint

## API Endpoint

- **GET** `/api/v1/audit/` - Retrieve all audit log entries

## Usage

The OpenAPI specification can be used to:

1. Generate API client code
2. Validate API responses
3. Document API behavior for backend developers
4. Test API endpoints

## Contract Details

See `audit-api.yaml` for complete specification including:
- Request/response schemas
- Authentication requirements
- Error response formats
- Example requests and responses
